TYPES OF LIST AND THEIR USES

In html, there are three ways to list items in an html file,
They are:

  • Unordered List
  • Ordered List
  • Definition List
Unordered List
This type of list is used to represent list with no positon.
Ordered List
Ordered list is used to represent items in numeral order.
Definition List
In definition list,listed items have text under the list ,giving more meaning to the item listed.
